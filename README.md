# QuickCommerce360: Enterprise Quick Commerce Data Intelligence Platform

## Business Problem

Quick commerce companies need real-time visibility into customer behavior, delivery performance, inventory optimization, and marketing ROI.

## Solution

Built an end-to-end analytics platform using Python, SQL, Data Warehousing, ETL Pipelines, and Power BI.

## Tech Stack

- Python
- Pandas
- SQL
- MySQL
- Power BI
- Data Warehousing
- ETL
- Git/GitHub

## Key Features

- Star Schema Data Warehouse
- Data Quality Framework
- ETL Audit Logging
- Customer Segmentation
- RFM Analysis
- Product Analytics
- Delivery Analytics
- Marketing Analytics

## KPIs

- Revenue
- Orders
- Customer Lifetime Value
- Average Order Value
- Delivery SLA
- ROAS
